Resúmenes inteligentes con ChatGPT. Control total de diseño, Google Fonts y administración dinámica de modelos.

== Description ==

Este plugin permite añadir un sistema de resúmenes de alta calidad mediante la API de OpenAI. Destaca por su panel de diseño que permite modificar el CSS del componente sin tocar una sola línea de código.

== Installation ==

1. Sube la carpeta a plugins.
2. Activa el plugin.
3. Configura la API Key en el menú de Ajustes.
