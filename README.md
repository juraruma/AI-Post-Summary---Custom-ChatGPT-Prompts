# AI Post Summary — Custom ChatGPT Prompts

A WordPress plugin that integrates ChatGPT summaries into any post 
with full aesthetic control — customize borders, shadows, typography, 
and spacing through a dynamic admin panel.

## Features

- ✅ Native OpenAI API integration
- ✅ Custom prompt editor in the WordPress admin panel
- ✅ Real-time model selection (GPT-4o, GPT-4o-mini, and more)
- ✅ Full visual customization: borders, shadows, fonts, and spacing
- ✅ Uses your own OpenAI API key — no middleman, no extra cost
- ✅ Optimized architecture for fast and elegant user experience

## Requirements

- WordPress 5.8+
- PHP 7.4+
- An active [OpenAI account](https://platform.openai.com/) with API access

## Installation

1. Download or clone this repository
2. Upload the plugin folder to `/wp-content/plugins/`
3. Activate the plugin from the WordPress admin panel
4. Go to **Settings → AI Post Summary**
5. Enter your OpenAI API key and customize your prompt
6. Open any post and generate your summary

## Configuration

- **API Key** — your OpenAI key (stored securely in your own server)
- **Model** — select the GPT model that fits your needs and budget
- **Prompt** — write your own instruction to control tone and format
- **Design panel** — adjust the visual appearance to match your theme

## Use Case

Perfect for bloggers, content creators, and WordPress site owners 
who want to add AI-powered summaries with a professional look — 
without depending on third-party subscriptions or SaaS tools.
